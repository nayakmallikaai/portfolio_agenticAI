# 🚀 Portfolio: Agentic AI System

An experimental **agentic AI portfolio project** showcasing how modern AI systems can go beyond simple prompts and operate as **autonomous, tool-using agents**.

---

## 🧠 What This Project Is

Traditional AI apps = input → output  
This project = **thinking system**

It builds an **agentic workflow** where:
- The AI decides what to do
- Chooses tools dynamically
- Executes multi-step tasks
- Produces structured, useful outputs

---

## ⚙️ Features

- 🤖 Agent-based architecture  
- 🔁 Multi-step reasoning (not just single prompt calls)  
- 🛠️ Tool usage (APIs / external data)  
- 🧩 Modular design (easy to extend agents)  
- 🧪 Experimentation with LLMs (Anthropic / OpenAI / others)  
- 📊 Structured outputs (not raw text dumps)
